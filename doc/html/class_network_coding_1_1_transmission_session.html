const bool TransmissionSession::SendPing ( )

Send ping packet to the remote host.

Returns
true : Reschedule ping transmission.
false : Stop sending ping.
  1. Check last pong time. If we have not received pong from remote host more than CONNECTION_TIMEOUT then we trigger disconnection for this session and stop sending ping.
  2. Send ping packet to remote host.
  3. Schedule next ping transmission.
